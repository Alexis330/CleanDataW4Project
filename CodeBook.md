## Getting and Cleaning Data - W4 Project

### Modifications

#### The original data is modified by 

* Merging the training and the test data sets to create one data set.
* Extracting only the measurements on the mean and standard deviation for each measurement. 
* Renaming variables to make them understandable.
* labeling the data.
* Creating a second, independent tidy data set with the average of each variable.
